MUTASI CRUD MENU USER GUIDELINES

        CRUD merupakan singkatan dari create, read, update, and delete digunakan untuk menggambarkan konvensi user interface yang memfasilitasi user untuk melihat, mencari, dan mengubah informasi pada data menggunakan form dan laporan berbasis komputer. Pada database mutasi berbasis CRUD menu, segala aktivitas untuk menambah, mencari, mengedit, hingga menghapus data dilakukan pada menu Mutasi agar lebih sistematis

  1. Akses Aplikasi
  1. New user

        Setelah menambahkan email new user pada opsi sharing dan menekan menu Mutasi - Load Data, maka akan dilanjutkan pada pop up otorisasi untuk keperluan akses pada program. Langkah-langkah yang perlu dilakukan adalah:

  • Pada pop up Authorization Required, pilih Continue

  • Kemudian pilih akun email yang digunakan untuk melanjutkan pada program

  • Pada pop up Google Hasn’t Verified This App, klik Advanced dan Go To (Unsafe)

  • Klik Allow pada pop up Wants To Access Your Google Account

  1. Existing user

        Sementara untuk pengguna yang sudah pernah akses pada program, langkah yang perlu digunakan untuk mengakses database hanya dengan klik menu Mutasi pada menu bar (posisi kanan atas) kemudian Load Data.

  1. Create

Program ini memiliki dua tab diantaranya tab Search dan Input. Kegiatan create atau menambahkan data mutasi dilakukan pada tab Input. Pada tab ini sudah disesuaikan format input data dengan uppercase untuk kesamaan format dan beberapa field form sudah disertai dengan dropdown list untuk mempermudah user dalam penginputan data.

  1. Read

        Proses pembacaan data (read) dapat dilakukan dengan mencari data yang diperlukan pada tab Search. Fitur pencarian dilakukan berdasarkan nama pengusul mutasi. User dapat langsung mengetikkan nama atau inisial dari pengusul mutasi di kolom pencarian. Pada tab ini, hanya beberapa data yang ditampilkan sesuai dengan urutan data yang paling sering dicari guna mempermudah user dalam melihat data. Data seperti persuratan dan tanggal surat tidak ditampilkan di tab pencarian. Kolom record di samping kanan atas menjelaskan jumlah pengusul dengan kata kunci yang dicari.

  1. Update

        Kegiatan update atau edit data dilakukan pada tab Search dengan mencari terlebih dahulu data pengusul berdasarkan kata kunci yang dimasukkan. Pada program ini, fitur pencarian dapat dilakukan berdasarkan nama pengusul atau cabang dinas pengusul.

Pada bagian ini, seluruh data yang sebelumnya sudah terisi akan secara otomatis ditampilkan di masing-masing field form. Selanjutnya user dapat mengedit data yang salah/belum terisi sesuai dengan data yang seharusnya.

Selanjutnya user dapat menyimpan perubahan data yang telah dilakukan dengan mengklik tombol Save. Sementara jika perubahan tersebut tidak jadi dilakukan dapat klik tombol Cancel Changes dan nantinya akan kembali ke halaman pencarian.

  1. Delete

        Sama seperti fitur edit, fitur delete data juga dapat ditemukan di tab pencarian dengan mencari terlebih dahulu nama pengusul mutasi. Bedanya, ketika user mengklik tombol delete, data tidak akan langsung terhapus. Melainkan akan ada tombol konfirmasi terlebih dahulu. Jika yakin untuk menghapus, maka user dapat menekan tombol Confirm, sementara jika tidak jadi menghapus data dapat dilakukan dengan klik tombol Cancel dan halaman akan kembali ke hasil pencarian seperti sebelumnya.
